# 数据库应用程序设计
一个简单的数据库应用程序项目设计,分为python开发的windows窗口程序和使用java和easyui模板制作的web端应用

数据库系统: mysql5.7

windows窗口程序: 使用python3.7,基于tkinter实现的GUI界面，pymsql建立数据库连接
